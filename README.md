# Python AI Engineering: The Java Developer's 30-Day Masterclass

This repository contains the syllabus and resources for Java developers transitioning to AI engineering, specifically optimized for **macOS (Apple Silicon)**.

## 📅 Roadmap Overview

### 📘 Phase 1: Pythonic Foundations (Days 1-7)
- **Goal:** Unlearn Java verbosity. 
- **Topics:** Comprehensions, `@dataclass`, Decorators, Context Managers, and Generators.
- **Project:** CLI Task Manager.

### 📊 Phase 2: Data Engineering (Days 8-14)
- **Goal:** Move from Row-thinking to Vector-thinking.
- **Topics:** NumPy Vectorization, Pandas DataFrames, Exploratory Data Analysis (EDA).
- **Project:** Kaggle Data Cleaning Pipeline.

### 🤖 Phase 3: Traditional ML (Days 15-20)
- **Goal:** Master the ML Lifecycle.
- **Topics:** Scikit-Learn, Regression, Random Forests, Model Evaluation.
- **Project:** House Price Predictor / Disease Classifier.

### 🧠 Phase 4: Apple Silicon Deep Learning (Days 21-25)
- **Goal:** Native ML training with Apple MLX.
- **Topics:** Tensors, Backpropagation, CNNs, Metal GPU utilization.
- **Project:** FashionMNIST Image Classifier.

### 🚀 Phase 5: GenAI & RAG (Days 26-30)
- **Goal:** Build Local AI Agents.
- **Topics:** Ollama, LangChain, Vector DBs (ChromaDB), RAG.
- **Capstone:** **Java-Source-Explorer** — A RAG system that analyzes your local Java codebases.

## 🛠 Setup
1. Install [Homebrew](https://brew.sh/).
2. Run `brew install python git ollama`.
3. Recommended IDE: VS Code with Python + Jupyter Extensions.
